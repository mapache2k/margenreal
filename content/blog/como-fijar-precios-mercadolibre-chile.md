La forma más común de fijar precios en MercadoLibre es buscar lo que cobran los competidores y poner un precio similar o ligeramente menor. El problema es que no sabes si los competidores ganan plata. Puedes estar copiando un precio que destruye el margen de todo el que lo usa.

## El enfoque correcto: de adentro hacia afuera

El precio se fija empezando por el costo, no por la competencia. La competencia informa el precio de mercado, pero no determina si puedes vender con margen a ese precio.

### Paso 1: Calcula tu costo total por unidad

```
Costo total = Costo de compra + Embalaje + Proporcional de operación
```

No uses solo el precio que pagas al proveedor. El embalaje, las etiquetas, el tiempo de preparación, los reclamos — todo tiene un costo real que se paga con el margen.

### Paso 2: Calcula la comisión efectiva de tu categoría

```
Comisión efectiva = Comisión de categoría ML × 1,19
```

El IVA 19% sobre la comisión es un descuento real. No lo ignores.

### Paso 3: Define el margen mínimo que necesitas

Esto depende de tu negocio, pero como referencia:
- Menor a 15%: zona de riesgo (cualquier variación de costos te vuelve negativo)
- 15–25%: viable pero ajustado
- Mayor a 25%: zona cómoda para absorber imprevistos

### Paso 4: Calcula el precio mínimo

```
Precio mínimo = (Costo total + Costo de envío) ÷ (1 − Margen mínimo − Comisión efectiva)
```

Este es el piso. Publicar por debajo de este número significa perder dinero.

### Paso 5: Comparar con el precio de mercado

Ahora mira a la competencia. Si tu precio mínimo está por debajo del precio de mercado, tienes espacio para competir con margen. Si está por encima, hay un problema estructural: costo demasiado alto, categoría equivocada, o producto que no puede venderse rentablemente en ML a ese precio de mercado.

## Ejemplo completo

**Producto:** mochila  
**Costo de compra:** $22.000 CLP  
**Embalaje + operación:** $1.500 CLP  
**Costo total:** $23.500 CLP  
**Categoría:** Deportes (13% comisión)  
**Publicación:** Premium, paquete ~800g ($4.590 envío)  
**Margen mínimo objetivo:** 20%

```
Comisión efectiva = 13% × 1,19 = 15,47%
Precio mínimo = ($23.500 + $4.590) ÷ (1 − 0,20 − 0,1547)
Precio mínimo = $28.090 ÷ 0,6453
Precio mínimo = $43.530 CLP
```

Si el precio de mercado para esa mochila está en $42.000–$45.000, puedes publicar a $44.990 con un margen real del 20%.

Si el mercado está en $35.000, no puedes competir con margen a ese precio. La decisión es subir el costo mínimo (no viable), reducir costos (negociar con proveedor), o buscar otro producto.

## Cuándo actualizar el precio

El precio no es estático. Revisa cuando:
- Cambie el precio del proveedor
- ML modifique las comisiones de tu categoría
- El precio de mercado suba o baje más del 10%
- Cambie el tipo de publicación (Clásica a Premium o viceversa)
