En lugar de teoría, tres ejemplos concretos con productos reales en categorías distintas. Todos calculados con las comisiones actuales de MercadoLibre Chile, IVA 19% incluido, publicación Premium.

## Ejemplo 1: Audífonos inalámbricos (Electrónica)

**Datos:**
- Costo del producto: $28.000 CLP
- Precio de publicación: $59.990 CLP
- Categoría: Electrónica (8% comisión)
- Publicación: Premium, paquete ~300g

| Concepto | Monto |
|---|---|
| Precio de venta | $59.990 |
| Comisión ML 8% | −$4.799 |
| IVA 19% sobre comisión | −$912 |
| Envío Premium (~300g) | −$3.990 |
| **Ingreso neto** | **$50.289** |
| Costo producto | −$28.000 |
| **Ganancia neta** | **$22.289** |
| **Margen** | **37,2%** |

Buen margen. La categoría de electrónica tiene comisión baja (8%) y el paquete liviano abarata el envío.

## Ejemplo 2: Ropa deportiva (Ropa y accesorios)

**Datos:**
- Costo del producto: $12.000 CLP
- Precio de publicación: $28.990 CLP
- Categoría: Ropa (15% comisión)
- Publicación: Premium, paquete ~400g

| Concepto | Monto |
|---|---|
| Precio de venta | $28.990 |
| Comisión ML 15% | −$4.349 |
| IVA 19% sobre comisión | −$826 |
| Envío Premium (~400g) | −$3.990 |
| **Ingreso neto** | **$19.825** |
| Costo producto | −$12.000 |
| **Ganancia neta** | **$7.825** |
| **Margen** | **27,0%** |

Margen razonable, pero la comisión de 15% en ropa es la más alta del mercado. Con un precio bajo, el envío representa una proporción importante.

## Ejemplo 3: Herramienta eléctrica (Herramientas)

**Datos:**
- Costo del producto: $35.000 CLP
- Precio de publicación: $55.000 CLP
- Categoría: Herramientas (12% comisión)
- Publicación: Premium, paquete ~2kg

| Concepto | Monto |
|---|---|
| Precio de venta | $55.000 |
| Comisión ML 12% | −$6.600 |
| IVA 19% sobre comisión | −$1.254 |
| Envío Premium (~2kg) | −$5.490 |
| **Ingreso neto** | **$41.656** |
| Costo producto | −$35.000 |
| **Ganancia neta** | **$6.656** |
| **Margen** | **12,1%** |

Margen ajustado. El paquete pesado encarece el envío y la diferencia precio-costo no es suficiente. Para mejorar: subir el precio a ~$65.000 o cambiar a publicación Clásica.

---

## Lo que muestran los tres ejemplos

1. La categoría importa — electrónica (8%) vs ropa (15%) hace una diferencia de 7 puntos de comisión
2. El peso del paquete puede destruir el margen en productos de precio medio
3. No todos los productos son rentables en Premium — hay que calcular antes de publicar
